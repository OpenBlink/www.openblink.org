# 🌟 M5 Stamp S3 つかいかた

## 💡 LED（ひかり）

### LED.set(color)
なかの LED（ひかり）の いろを かえます。

- `color` - いろの すうじ [あか, みどり, あお]、0から255まで
- れい：`LED.set([255, 0, 0])` - あかく ひかる
- れい：`LED.set([0, 255, 0])` - みどりに ひかる
- れい：`LED.set([0, 0, 255])` - あおく ひかる
- れい：`LED.set([255, 255, 255])` - しろく ひかる
- れい：`LED.set([0, 0, 0])` - ひかりを けす

## ⏱️ タイマー

### sleep(seconds)
プログラムを すこし とめます。

- `seconds` - まつ びょうすう（0.5 みたいな すうじも つかえます）
- れい：`sleep 1` - 1びょう まつ
- れい：`sleep 0.5` - 0.5びょう まつ

### sleep_ms(milliseconds)
プログラムを すこし とめます（ミリびょうで きめます）。

- `milliseconds` - まつ ミリびょうすう
- れい：`sleep_ms 100` - 100ミリびょう まつ
- れい：`sleep_ms 1000` - 1びょう まつ
